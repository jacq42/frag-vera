This is an [express](https://expressjs.com/) project bootstrapped with [`express-generator`](https://expressjs.com/en/starter/generator.html).

Add [nodemon](https://www.npmjs.com/package/nodemon) to start in development mode.

Express with typescript: https://www.bezkoder.com/express-typescript-example/

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
